# 🔒 CyberSlacker 隐私政策

我们非常重视您的隐私。本软件（CyberSlacker 赛博摸鱼员）的设计理念是“绿色、透明、无公害”。

### 1. 数据收集
本软件**不会**收集、存储或上传您的任何个人身份信息（PII）。

### 2. 外部接口调用
为了提供准确的倒计时功能，本软件会定期访问以下公开 API：
- **Timor.tech**: 仅用于获取法定节假日及调休数据。
此过程中不会发送您的个人设备 ID 或任何敏感信息。

### 3. 本地存储
用户的配置信息（如下班时间、发薪日、窗口位置等）均存储在您本机的 Windows 配置文件中。我们无法远程访问这些数据。

### 4. 第三方链接
软件内部提供的 GitHub 链接仅用于问题反馈和版本检查。
